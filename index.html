Experience

BestDay. (2017 - Actual)

Programmer focused on developing and maintenance of applications based primarily in: Android(Java native). A sales point used by sales representatives to offer tours and other activities to the passengers in hotels all around Cancún, Riviera Maya, Puerto Vallarta, Los Cabos and Punta Cana. The application has a main functionality of working online and offline, letting the representatives sell even when they have no internet on their tablet, something very common in the Riviera Maya Hotels. The sales representatives is bound to an specific tablet, so he can only use the one that was assigned to him. In order to work completely offline the applications uses a synchronization method to retrieve the information that it need from a service and store it in the database using greenDao as ORM. The application also has the functionality of print coupons, voucher and transfer vouchers making us of a bluetooth printer.

Maintenance of the web service that feeds with all the necessary information to the Mobile application. Built using .Net standar, .Net Full Framework, Entity Framework as ORM, Linq, ASP .Net, ADO .Net and Web API 2. Maintenance of the SQL Server database mainly on all the tables, function and stored procedures that are consume by the Web Service. Continuous integration with Azure DevOps. Usage of Git Flow methodology. Application of Kanban as the agile work methodology.
